- Python 3.9.6

## START

### 建立虛擬環境
```zsh
python -m venv tutorial-env
```
### 進入 venv 虛擬環境
```zsh
source myenv/bin/activate
```
### 離開虛擬環境
```zsh
deactivate
```

## 安裝 python 套件
```zsh
pip3 install torch numpy matplotlib pillow torchvision torchsummary torchaudio
```
