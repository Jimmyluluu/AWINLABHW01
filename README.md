# AWINLab HomeWork

## 環境版本
- Python 3.8.10
- Ubunbu 20.04
- MacOS

## 訓練使用硬體
- R5-5600x
- RTX3070Ti-8G

## START HW01

### 建立虛擬環境
```zsh
python -m venv hw01env
```
### 進入 venv 虛擬環境
```zsh
source hw01env/bin/activate
```
### 離開虛擬環境
```zsh
deactivate
```

## 安裝 python 套件
```zsh
pip3 install torch numpy matplotlib pillow torchvision torchsummary torchaudio openpyxl pandas
```
